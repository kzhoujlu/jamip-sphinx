class jamip.compute.pool.Pool(pool=None, **kwargs)

Bases: object

任务池是存储计算输入数据，并将根据任务提交请求提交和记录任务状态。 任务池分为数据部分(jp.pool)和记录部分(jp.dat/jp.dir/jp.bak) 其中数据部分基于pickle，记录部分基于dbm 任务池整体支持轻量级的并行访问，安全性有待测试，可能通过冗余的方式解决

Atributes:

_pool: Calculation data saved in *.pool __db: Work status of jobs, saved in *.dat
